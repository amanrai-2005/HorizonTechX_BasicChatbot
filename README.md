# HorizonTechX - Basic Chatbot

## Project Description
This project is a simple rule-based chatbot developed using Python. The chatbot responds to predefined user inputs and demonstrates the use of basic Python programming concepts.

## Features
- Responds to greetings such as "hello"
- Answers simple questions like "how are you"
- Handles user appreciation messages
- Ends conversation when the user types "bye"

## Technologies Used
- Python 3

## Concepts Used
- Functions
- Conditional Statements (if-elif-else)
- Loops
- User Input and Output
- String Manipulation

## Sample Interaction

You: hello

Bot: Hi! How can I help you today?

You: how are you

Bot: I'm fine, thanks!

You: bye

Bot: Goodbye! Have a great day!
